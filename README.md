# Mind App — Dual N-Back Trainer

Projet web **vibe coded** pour une utilisation personnelle (entraînement cognitif type Dual N-Back).

## Objectif
- Entraîner la mémoire de travail avec un mode position + audio.
- Suivre la progression via un dashboard simple.

## Statut
- Projet personnel / expérimental.
- Pas un dispositif médical.
- Les résultats sont indicatifs.

## Stack
- HTML, CSS, JavaScript
- PWA (manifest + service worker)
- Hébergement statique (Netlify)

## Confidentialité
- Les données de session sont stockées localement dans le navigateur (`localStorage`).
- Aucun backend personnel actif côté PC.

## Développement local
```bash
python -m http.server 8080
```
Puis ouvrir: `http://localhost:8080`
